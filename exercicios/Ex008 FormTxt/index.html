Principais formatações - Parte 2

Negrito/Destaque

Primeiro exemplo é usando a tag b (forma não semântica).

Segundo exemplo é usando a tag strong (forma semântica).

Itálico/Enfase

Primeiro exemplo é usando a tag i (forma não semântica)

Segundo exemplo é usando um atalho ctrl+shift+p

Terceiro exemplo é usando a tag em (forma semântica)

Texto marcado

Para marcar um texto, use a tag mark

Também posso marcar um texto usando a tag + um estilo em css.

Texto grande e pequeno

Para texto grande use uma formatação que aumente o tamanho da fonte e para texto pequeno use a tag small

Texto deletado

Para indicar um texto a ser lido, mas não considerado, use a tag del

Texto inserido

Para indicar um texto que foi adicionado depois use a tag ins

Texto sobrescrito

Utilizado para criar texto como no exemplo x20+3 utilizando a tag sup

Texto subscrito

Utilizado para criar texto como no exemplo H2O utilizando a tag sub
